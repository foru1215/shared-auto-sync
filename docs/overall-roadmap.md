# Overall Roadmap

## この計画は何をするものか

この計画は、24か月で次の 5 つを作る計画です。

1. 資格の土台
2. 制御・図面・PLC の理解
3. 実務で語れる証拠
4. AI を使った成果物
5. 転職で出せる書類と説明材料

つまり、`資格` だけでも `AI` だけでもなく、
`電気 + 制御 + 実務証拠 + AI + 転職準備`
を 1 本のストーリーにする計画です。

## 最終的に手に入れたい状態

2028-03 のゴールはこれです。

- 第一種電気工事士を軸にした資格の土台がある
- 制御基礎、図面読解、PLC の学習成果がある
- 発電機盤・ポンプ盤・社内改善などの実務証拠がある
- AI / Codex / Claude Code を使った成果物がある
- 職務経歴書、面接ストーリー、ポートフォリオが揃っている

## 何を順番にやるのか

この計画は、次の順番で進みます。

1. まず資格の主軸を取る
2. 次に制御基礎と図面読解を固める
3. その上で PLC に進む
4. 並行して実務証拠を残す
5. その後 AI 成果物を強化する
6. 最後に転職用の形へ統合する

## つまり今後ずっと何をやればいいのか

毎 phase でやることは同じです。

1. その phase の主役を 1 つ決める
2. 主役に必要なノート / 成果物 / 証拠を作る
3. 週次レビューでズレを直す
4. 次 phase に持ち越すものを整理する

## 何を見れば迷わないか

迷ったら、毎回この順で見れば足ります。

1. `Phase Card`
2. `Win Condition`
3. `Issue`

Issue を全部見る必要はありません。
まず見るべきなのは「今の phase の主役」と「その phase の勝ち条件」です。

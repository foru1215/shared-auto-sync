# 24か月キャリア戦略計画 — 要約版

**対象期間**: 2026/3/14 〜 2028/3/31
**最終更新**: 2026/3/14

---

## 全体目的

制御盤メーカー営業から **「制御・設備 × AI の実務人材」** へ 24か月で転換する。
資格で基礎信用を獲得し、制御・PLC の実務理解で技術者と対等に会話できる力を積み、Codex / Claude Code で差別化する。

---

## 優先順位

1. 将来の年収の伸び
2. 役職者ルートへの接続
3. 現職からの接続性
4. 実現可能性
5. GitHub Projects で管理しやすいこと
6. 毎月・毎週迷わない運用
7. 学習内容が転職で使える証拠になること
8. 学歴不足を資格・実務証拠・AI 活用で埋めること
